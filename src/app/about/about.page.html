About

IoT Hub Stub is a small, simple open source application that allows one to quickly interact with an Azure IoT Hub solution. The IoT Hub Stub eliminates the requirement of a micro-processor board (e.g. Raspberry Pi, Arduino, etc.) to transmit real-time data. Iot Hub Stub transmits sensory data from your mobile device.
